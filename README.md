# NodeMCU_ESP8266_MultipleSensors
<h3>Description</h3>
<span>IoT Project</span>
<p>
  Card: NodeMCU ESP8266 platform.       <br>
  * MLX90614 - Temperature Sensor.      <br>
  * MQ135 - Gas Sensor.                 <br>
  * LDR   - LDR Sensor.                 <br>
</p>
